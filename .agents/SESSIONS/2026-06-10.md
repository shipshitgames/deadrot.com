   `import.meta.glob` — `bun test` cannot load it. Migration attempted, 3
   suites failed, backed out per plan fallback. Don't retry without removing

   coined boss names (Maw Shepherd, Hull Chorus, Junction Knell) are
   `coined: true` + "(provisional)" in the vault until promoted. Documented in
   `Universe/Vault-Conventions.md`.

   by epoch/tick) so server + every client pick the same beat — packages/warline
   stays dependency-free (timelineRefs are strings, pinned by an app-side test).
5. **War record is display-only v1** — localStorage only aggregates on the
   prod same-origin hub; PartyKit sim never reads it.
6. **Orchestration pattern**: implement in isolated worktrees → quality gates →
   PR → independent adversarial reviewer per PR. The reviewer **failed #320**
   (redline defeats unrecorded — implementer's "no defeat state" claim was
   false) and found pactfall's abilities unreachable on touch; both fixed

   the 5 stacked PRs (GitHub did NOT retarget). Recovery: re-push the base ref
