# Sprite Generation Ledger Template

Use this shape inside `lore/Art/Generation-History.md`.

```markdown
## YYYY-MM-DD - <asset-id> - v##

- Status:
- Game:
- Faction:
- Character/role:
- View:
- Tool:
- Plan:
- Kind:
- Source output:
- Workspace draft source:
- Workspace draft cutout:
- Final asset:
- Prompt source:
- Post-processing:
- Notes:
- Decision:
```

Decision values:

- `keep-direction`: good enough to develop further.
- `reject`: not useful; include why.
- `regenerate`: useful target but prompt drifted.
- `promote-next`: approved for `sprite-asset-promotion`.
