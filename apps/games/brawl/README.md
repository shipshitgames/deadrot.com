# Brawl

Brawl is the first Deadrot one-on-one battlefield fighting game prototype. It
turns the Warline front into a character-scale duel: pick a faction fighter,
let the game select the rival, then fight a single arcade round.

## Commands

- `bun run dev`
- `bun run typecheck`
- `bun run test:unit`
- `bun run build`

Runtime sprites currently reuse existing `@shipshitgames/assets` entity plates.
The production fighting-game sprite pipeline is tracked by the Brawl epic.
