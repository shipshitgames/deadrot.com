# Design Locks

The lore vault owns the visual truth. Games decline these locks into their own
camera, scale, and runtime formats.

Use this folder for per-subject visual design locks:

- `Design/Bestiary/<subject>-DESIGN.md`
- `Design/Locations/<subject>-DESIGN.md`
- `Design/Characters/<subject>-DESIGN.md`

Each design lock must point back to the canon page and [[Style-Bible]], then
declare the approved silhouette, material/color grammar, locked source poses,
game adaptations, and validation status.

Runtime assets in `packages/assets/games/...` are derivatives. They do not
define lore canon.
