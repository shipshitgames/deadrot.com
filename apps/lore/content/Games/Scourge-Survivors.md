# Scourge Survivors

**At a glance:** first-person horde-survivors shooter · survive the swarm — drop into a breach, grind the Scourge, draft upgrades, push deeper until you're overrun · lone [[The-Pyre|Pyre]] operator (offense; burn the source) · the offensive front of the [[War-for-the-Lanes]], where [[Deadlane]]'s Wardens can only hold the line — Resistance-era, after the fall in [[Zero-Day]].

The flagship FPS. A lone [[The-Pyre|Pyre]] operator drops into a breach and must **survive
the Scourge** — endless, growing swarms — in first person. The Vampire-Survivors loop
(auto-scaling weapons, XP, level-up drafts, orbit / bolt / nova) with DOOM-fast gunfeel
and gore.

## Core loop (survivors-centric)
The Survivors mode **is the game**, not a side mode: drop in → get swarmed → grind the
[[Scourge]] → level up → draft upgrades → push deeper room to room until you're overrun.
Campaign / multiplayer become secondary or fold into this loop.

This is the flagship implementation of the shared [[Survivors-Loop]].

## First Sprite Roster

- Scourge enemies: [[Swarm-Ripper]], [[Swarm-Spitter]], [[Breach-Boss]].

## Canon role
The offensive front (see [[Premise]]). You go into the breach where [[Deadlane]]'s
[[The-Wardens|Wardens]] can only hold the line. The deepest region is [[Perdition]].

## Maps
Many breach-arenas with rooms and levels — each a chamber of the breach (see the map set).

## Project
The `games/scourge-survivors` folder is the working game project, focused on the Survivors
core and expanding with more maps / levels / rooms.
