# Survivors Loop

The studio can reuse a Vampire-Survivors-style run structure across very different games
without making the games feel identical.

## Core Shape

- Escalating waves or pressure.
- Drops that pull the player forward: XP, scrap, blood, salvage, gold.
- 1-of-3 upgrade drafts during a run.
- Build synergies that stack into overpowered combinations.
- Bosses or objectives that gate the run.
- Persistent meta-upgrades between runs.

## Why It Fits Canon

The Scourge escalates. Breaches pulse. Lanes collapse under pressure. Every game can
express that pressure differently while sharing the same addiction loop.

## Per Game

- [[Scourge-Survivors]] — FPS Purger buildcraft: guns, orbitals, nova, bolts, survivability.
- [[Deadlane]] — tower-defense buildcraft: towers, walls, economy, commander powers.
- [[Pactfall]] — MOBA buildcraft: hero perks, lane objectives, Scourge surge responses.
- [[Starblight]] — arcade pilot buildcraft: weapons, drones, shields, orbital boss phases.

## Rule

The loop is design DNA, not a template prison. Controls, camera, fantasy, and pacing should
change per title; escalation, drafts, synergies, and meta-progression create cohesion.
