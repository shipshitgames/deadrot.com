# Shared FX

Game-agnostic DOOM-grade combat FX, used identically by every Scourge-universe
game. Visceral, not neon — blood `#c1121f`, hellfire `#ff6a00`, gunmetal, bone.

First-pass placeholder slots:

- `blood-splatter.webp` — impact/gore decals and bursts.
- `ember-burst.webp` — hellfire embers and burn particles.
- `muzzle-flash.webp` — weapon muzzle flashes.
- `breach-glow.webp` — toxic Scourge breach glow / node pulse.

These are referenced from `assets-catalog.json` under `shared` (`kind: "fx"`).
