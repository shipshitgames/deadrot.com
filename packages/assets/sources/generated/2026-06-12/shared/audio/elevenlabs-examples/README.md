# ElevenLabs Shared Audio Examples

Audition-only batch generated from prompts in
`packages/assets/shared/audio/audio-catalog.json`.

- Tool: ElevenLabs Sound Effects API
- Model: `eleven_text_to_sound_v2`
- Date: 2026-06-12
- Scope: candidate replacements for code-generated shared audio and procedural
  fallback cues.

The `.mp3` files are preserved masters from the provider response. The `.webm`
files are local WebM/Opus transcodes for browser auditioning. Do not promote
these into `packages/assets/shared/audio` until one is selected and the target
runtime manifest entry is updated.
